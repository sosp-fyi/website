# Force of Static Friction
Occurs when an object is stationary on a surface, but it would like to slide, but the surface is stopping it from sliding
- Symbol: $f_s$
- Magnitude = depends on the situation, but has maximum possible value: $f_{\text{s,max}}$ = $u_s$ * N
- Direction opposes direction in which object would move if there was no friction
# Force of Kinetic Friction
Occurs when an object slides along a surface
- Symbol: $f_k$
- Magnitude = $f_k$ = $u_k$ * N
- Direction = opposes motion

$f_s$ = F
$f_s$ = $u_s$ * F $_N$
u = coefficient of static friction

$$
f_k = u_k F_N
$$
$u_k$ = coefficient of kinetic friction
