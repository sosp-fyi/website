# What is a Vector?
A mathematical object with magnitude (r) & direction
used to represent & quantify physical dimensions
- position
- velocity
- acceleration
- and more
Drawn as an arrow with length r, in a specific direction.

## Vectors and Scalars
- A scalar quantity is completely specified by a single value with an appropriate unit and has no direction.
- A vector quantity is completely described by a number (& appropriate units) plus a direction.

In 1-d (on a line) "x"
- scalar is always +
- vector "direction" is + or -
In 2-d (plane)
- scalar is always +
- direction is represented as an angle related to the x-axis

Cartesian coordinates
r = (x, y)

Polar coordinates
r = (r, $\theta$)

# Vectors Can Be Added
- Draw the first vector with tail at the origin of coordinates
- Draw the next vector with tail at the tip of the first vector
- repeat
- Sum (resultant) is drawn from tail of first to tip of last vector

# A Vector Can Be Multiplied by a Scalar (A)
Direction: same as original vector
Magnitude: product A × r

# Basic Vector Arithmetic
Vector properties (2-d example, works in higher-d)
A = (A $_1$, A $_2$) | (1,2) = (x, y) or (r, $\theta$)
B = (B $_1$, B $_2$)
- Equality A = b ↔ A $_1$ =B $_1$, A $_2$ = B $_2$
- Addition A + B
	- commutative A + B = B + A
	- associative A+(B+C) = (A+B)+C
	- negative → subtraction A-B=A+(-B)
- Multiplication
	- Scalar(k) x vector: B=kA
	- Vector x vector: more on this later

(R $_x$, R $_y$) = (A $_x$ + B $_x$, A $_y$ +B $_y$)

# Adding Vectors
- Graphical method is good for visualization, not quantitative
- Algebraic addition of separate Cartesian coordinates is straightforward but not really a vector operation

Constructing a vector from Cartesian coordinates
- A described by x, y coordinates = (A $_x$, A $_y$)
- x-component vector has a length A $_x$, coordinates (A $_x$, 0)
- y-component vector has a length A $_y$, coordinates (0, A $_{y}$)
A = A $_x$ + A $_y$

Define unit vectors along +x, +y directions
- x: x-axis, i
- y: y-axis, j
Dimensionless: has "length"=1, direction

## Component Vectors
Multiply each unit vector by corresponding coordinate
A $_x$ i = A $_x$
A $_y$ j = A $_y$

Example:
A = velocity
A $_x$, A $_y$ = same, velocity
components have the same units as vectors

A = A $_x$ i + A $_y$ j
