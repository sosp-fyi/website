Circular motion in polar coordinates
Particle at radius r, angle theta
- Change of angle → angular motion → angular speed
counterclockwise > 0
clockwise < 0
$\omega = \frac{d\theta}{dt}$
- Change of angular speed → angular acceleration

# Rigid Object under Constant Angular Acceleration
The mathematical relationships between angular in a plane are the same as 1-d linear (transitional)
position $\theta$ = position x
velocity $\omega$ = velocity $V_x$
acceleration $\alpha$ = acceleration $a_x$

$L = r \theta$
$v=r\omega$
$a_t = r\alpha$
