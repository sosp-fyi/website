# Energy Changes to a Gas
State variables: P, V, T, E $_\text{int}$
Transfer variables: work (W), heat (Q)

heat transfer and/or work change the internal energy:
$$
\Delta E_\text{int} = Q + W_{\text{on gas}}
$$

A. isolated system (static) - no work or heat transfer
$$
Q = W = 0 => \Delta E_\text{int} = 0
$$
B. non-isolate: state changes by work and/or heat transfer ↔ path on P vs. V graph in a closed cycle, the path is a loop (cyclic process) 
final state = initial state
$$
\Delta E_{\text{int}} = 0 => Q = -W
$$
- For ideal gas, E $_{\text{int}}$ is proportional to T:
$$
=> \Delta E_{\text{int}} = 0 <-> \Delta T = 0
$$

Isobaric: constant pressure
Isovolumetric: constant volume
Isothermal: $\Delta E_{\text{int}} = 0$
Adiabatic: $Q = 0$
