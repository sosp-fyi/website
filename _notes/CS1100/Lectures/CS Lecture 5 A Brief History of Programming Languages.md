# Types of Programming Languages
- Assembler (low level)
- High level languages
	- Procedural (90% of industry work)
		- Most common: Java, C++
	- Functional
		- LISP, ML, SCHEME
	- Logic Programming

### Speedcoding
- first HLL on IBM machines
- floating point numbers
- it was an interpreter
- used 30% of system memory
- ran 10-20 times slower than assembly language

Interpreted languages would execute the program on program at a time.
Compiled languages are converted into machine code then executed. Think compiling a C# file before being executed.

### FORTRAN Language is Born in 1954
FORmula TRANslating
Major emphasis on scientific programming

Apollo missions used FORTRAN IV
Dramatically changed the way computers were used forever.

### LISP 1959
List Processing = LISP
A functional language
- Designed at MIT by McCarthy
- Still the dominate language for AI (not anymore Python is used more)
- Pioneered functional programming
- Syntax is based on lambda calculus

- Contemporary dialects:
	- COMMON LISP, Scheme
- Object-oriented versions:
	- Flavors, LOOPS, CLOS
- Related languages:
	- ML, Miranda, Haskell

- Higher-order functions
	- Functions that take other functions as arguments

### ALGOL
Algorithmic language = ALGOL

- goals of the language
	- close to mathematical notation
	- good for describing algorithms
	- must be translatable to machine code

- Not meant to be implemented, but variations of it were mad jovial

### COBOL (1960)
common business oriented language = COBOL

- based on Flow-Matic, which had:
	- Names up to 12 characters, with embedded hyphens
	- English names for arithmetic operators
	- Data and code kept completely separate
	- Verbs as the first word in every statement
- COBOL Design Problems:
	- Arithmetic expressions?
	- Subscripts?
	- Fights among manufacturers
- Contributions:
	- Macros
	- Hierarchical data structures
	- Nested selection statements
	- Long names with hyphens
	- Separate data division
