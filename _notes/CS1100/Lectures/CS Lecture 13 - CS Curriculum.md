# Pre-Junior / Junior Years
## First Semester
- General education course
- Comm1071: introduction to effective speaking
- CS4071: Design and analysis of algorithms
	1. Role of algorithm in computing
	2. Divide and conquer algorithms
	3. Sorting complexity, heap sort, quick sort
	4. Binary search trees, red-black trees
	5. Dynamic programming
	6. Greedy algorithms
	7. Elementary graph theory, minimum spanning trees, single source the shortest paths, all pair shortest paths, maximum flow
	8. Polynomials and FFT
	9. NP completeness
- CS4092: Database design and development
	1. Architecture and usage of DBMSs
	2. Physical storage: file structures, indexing, hashing (static and dynamic)
	3. Entity-relationship model: application design
	4. Relational model: schema development and query specification
	5. Formal query languages (relational algebra and tuple calculus)
	6. Query optimization (logical and physical)
	7. Functional dependencies and normalization
	8. Transactions: recovery and concurrency
	9. XML data modeling and web-based database design and development
- EECE3093C: Software engineering
	- Introduce various approaches and methodologies for software development
	- Different phases of software development lifecycle, including:
		- Requirements analysis
		- Specification
		- Software design
		- Software construction
		- Software testing
		- Software maintenance
	- A project and team-based class, has programming lab hours
