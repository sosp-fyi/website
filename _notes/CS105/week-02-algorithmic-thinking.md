We introduced algorithm design using step-by-step problem decomposition.

## Key Points

- Break large tasks into smaller deterministic steps.
- Use conditionals for branching behavior.
- Test with normal, edge, and invalid inputs.

## Mini Example

To find a missing number in a sequence, sort first, then scan for the first gap.
