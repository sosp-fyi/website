### Significant Figures

$5300.$ has 4 significant figures

### Significant Figures in Calculations
1. Multiplication and division:
answer has same number of significant figures as the value with the fewest significant figures

Example:
Multiply the following numbers
$9.2 cm * 6.8 cm * 0.3744 cm = 23.4225 cm^{3} = 23 cm^{3}$ 

2. Addition and subtraction:
Answer has same number of decimal places as the value with the fewest decimal places.

Example:
$83.5mL+23.28mL = 106.78mL = 106.8 mL$

$865.9 mL - 2.8121 mL = 863.0879 mL = 863.1 mL$ 

$(3.8621 * 1.5630) - 5.98 = 6.0365 - 5.98 = .0565 = .06$

### Precision, Accuracy, and Error
Precision - how close measurements are to each other.
Accuracy - how close measurements are to actual value.
Systematic error - values either all higher or all lower than actual value.
(part of the experimental system)
Random error - values both higher and lower than actual value.

### Definitions
Element - consists of only one type of atom. It cannot be broken down into simpler substances by physical or chemical means.
Molecule - consists of two or more atoms chemically bound together, behaving as an independent unit.
Compound - composed of two or more elements, chemically combined. (All compounds are molecules but not all molecules are compounds).
Mixture - two or more elements and/or compounds, physically intermingled. 

### General Features of the Atom
* Spherical
* Electrically neutral (same # of electrons and protons - unless the atom is actually a positive or negative ion!)
* Positively charged nucleus surrounded by negatively charged electrons
* Nucleus consists of protons and neutrons
* Mostly space!

Isotopes: atoms of an element with the same number of protons, but a different number of neutrons
isotopes have the same atomic number, but a different mass number.

### Atomic Symbol, Number and Mass
$\frac{A}{Z}X$
X - atomic symbol of the element
A - mass number; A = Z + N
Z - atomic number (the number of protons in the nucleus)
N - number of neutrons in the nucleus

### Calculating Average Atomic Mass
atomic mass = ∑ (isotopic mass) * (fractional abundance of isotopes)
for example:
the natural abundance of carbon isotopes is 98.9% $^{12}$C and 1.1% is $^{13}$C.
atomic mass = $(12 amu) * (0.989) + (13 amu) * (0.011)$
$= 11.868 amu + 0.143 amu$
$= 12.011 amu$

Example:

Chlorine has two naturally occurring isotopes, $^{35}$Cl and $^{37}$Cl Calculate the (average) atomic mass of chlorine.

$\frac{(34.96885 + 36.96801)}{2} = 35.96843 amu = 35.97 amu$
